Resume Analyzer

An AI-powered Resume Analyzer built with Flask and Gemini AI that helps job seekers improve their resumes by providing ATS scores, detailed feedback, and actionable recommendations.

-Features
Upload PDF resumes
Extract resume content automatically
AI-powered resume analysis using Gemini
ATS compatibility score
Skills and keyword evaluation
Personalized improvement suggestions
User authentication system
Admin dashboard for management
Responsive and modern UI
Feedback submission system
-Tech Stack
Frontend: HTML, Tailwind CSS, JavaScript
Backend: Python, Flask
Database: SQLite / PostgreSQL
AI: Gemini API
Authentication: Flask-Login
-Installation
-Clone the repository
git clone <repository-url>
cd resume-analyzer
-Create a virtual environment
python -m venv venv
-Activate the environment
venv\Scripts\activate
-Install dependencies
pip install -r requirements.txt
-Configure environment variables
SECRET_KEY=your_secret_key
GEMINI_API_KEY=your_gemini_api_key
-Run the application
python app.py

Visit:

http://127.0.0.1:5000
Deployment

The application can be deployed on Render, PythonAnywhere, Railway, or any cloud platform supporting Flask applications.
