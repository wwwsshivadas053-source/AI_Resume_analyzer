Resume Analyzer

This application provides role-based authentication, PDF resume analysis, ATS scoring, feedback capture, and admin management for users, feedback, and analyzer logs.

Auth

Register/login with role-based redirects for users and admins.

Analyzer

Upload PDF, extract text, calculate ATS and JD match scores, then generate suggestions.

Feedback

Home-page feedback form stores submissions in SQLite.

Admin

CRUD controls for users, feedback, and analyzer logs.

UI

Responsive Tailwind cards, forms, gradients, and premium dashboard layouts.

Database

SQLite database seeded with an admin account for the shared login form.
